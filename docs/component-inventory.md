## UI primitives (`app/src/components/ui/`) — shadcn/ui

Layout & structure: `card`, `separator`, `sidebar` (with `SidebarProvider`, `SidebarInset`, `SidebarTrigger`, `SidebarMenu*`, `useSidebar`).

Form & input: `input`, `textarea`, `button`.

Display & feedback: `badge` (variants incl. `brand`, `danger`, `warning`, `secondary`, `outline`), `avatar`, `progress`, `sonner` (toast `Toaster`).

Overlays: `dropdown-menu`.

> Modals in both views are **not** a Radix Dialog — they are hand-rolled fixed-overlay `div`s with click-outside + Escape handling. Another consolidation candidate.

## Design system notes

- **Theming:** brand palette via CSS variables (`--brand`, `--brand-light`, `--brand-border`) defined in `globals.css`, referenced inline.
- **Icons:** `lucide-react` throughout; action buttons pair an icon with a label.
- **Composition:** `cn()` (`lib/utils.ts`) merges Tailwind classes; `class-variance-authority` drives variant styling in primitives like `badge`/`button`.
